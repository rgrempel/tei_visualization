The purpose of this web-site is to use Javascript to view TEI documents in interesting ways. For technical information, you can consult the source code, which is located at http://github.com/rgrempel/tei_visualization .

To get started here, use the "Documents" menu to either open a TEI document that is already here, or upload a new one of your own. Once the document is open, you can use the other menus to explore it in various ways. Most menus open a "window" of one kind or another ... you can use the buttons on the window header to move the windows into the left side, right side, or into a tab below.

The "panels" menu will show a list of windows you have currently open, and you can use it to switch among them.

Note that this site only works with Firefox at the moment. The code is heavily dependent on client-side XSLT, and is using XSLT features that only Firefox is supporting.

The site is a work-in-progress ... if you have difficulties, or suggestions, etc., contact Ryan Rempel, at rgrempel@gmail.com.

This software was created to support a course on the "Digital Word" at Canadian Mennonite University. Information about that course is available at a Moodle site at http://www.pauldyck.com.
